# Client app (tenant portal)

Trang web tra cuu thong tin phong cho nguoi thue.

## Cai dat

1. Mo terminal tai thu muc client: `cd client`
2. Cai dependencies: `npm install`
3. Chay dev: `npm run dev`
4. Build san pham: `npm run build`

## Tai khoan mau

- So dien thoai: `0901234567`
- Ma truy cap: `BA23-4589`

## Tinh nang

- Dang nhap bang so dien thoai + ma hop dong duoc quan ly cap.
- Tra cuu thong tin hop dong, dia chi, phong, nguoi quan ly.
- Xem tien phong, coc, tien ich va lich su thanh toan mau.
- Ghi chu huong dan lien he quan ly.
